## 👥 How to use the app
### Add employees
Open the employee section and enter the basic details for each worker. Save the record after you fill in the form.

### Create payroll entries
Pick an employee, enter the pay period, and add the amounts needed for that cycle. The app can help you keep each payroll run in order.

### Add bonuses
Use the bonus fields to record one-time payments or rewards. This helps keep base pay and extra pay separate.

### Review tax data
Check the tax section before you finish a payroll run. This helps you keep payments and deductions in line with your records.

### View reports
Open the reporting area to review payroll data. Reports help you check totals, track changes, and print or share records when needed.

## 🗃️ Data and storage

Keep your database backed up on a regular basis. That helps protect your payroll history if your PC has a problem or you move to a new machine.

## 🛠️ Common setup problems
### The app does not open
- Make sure you downloaded all files from the release or project page
- Check that the `.exe` file is in the extracted folder
- Run the app again after extraction

### Windows blocks the file
- Right-click the file and choose Properties
- If you see an unblock option, select it
- Try opening the app again

### The database does not connect
- Check the server name and database name
- Make sure PostgreSQL is running
- Check the user name and password
- Confirm that the port number matches your database setup

### Text or buttons look too small
- Change Windows display scaling
- Use a screen resolution that fits the app window
- Restart the app after changing display settings
